A compilação deste projeto é semelhante à compilação dos exercicios das aulas práticas

Instruções de compilação:
Método 1: Compilar o programa usando CMake no terminal ou na aplicação. Requer instalação válida do CMake.
Método 2: Abrir a pasta com os ficheiros do projeto como um novo projeto no CLion. Dar load ao ficheiro "CMakeLists". Correr o compilador do CLion. Requer instalação válida do Clion.
